# lab6it-15-16-2025gr8
Описание проекта: Угадай число Компьютер загадывает случайное число в заданном диапазоне, а игрок должен угадать его. После каждой попытки программа сообщает, больше или меньше загаданного числа предложенное игроком значение. Особенности реализации: Функции для генерации случайного числа, считывания ввода пользователя и проверки ответа. Основные элементы: Использование генератора случайных чисел и простых условных операторов.

Роли участников: Брындин Павел - написание функции, которая анализирует вводимое пользователем число и редактирование кода, в случае ошибок . Бабушкин Никита - написание функции, которая генерирует случайное число,  оформление основной функции. Карионов Дмитрий - создание тела основной функции и создание пользовательского интерфейса.

Инструкция по запуску:

1. Запустить программу lab6.cpp.
2. Задать диапазон выборки числа.
3. Приступить к попыткам угадывания числа.
4. Победа!
